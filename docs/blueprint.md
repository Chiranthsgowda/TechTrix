# **App Name**: TechTrix Tournament Tracker

## Core Features:

- Leaderboard Display: Display the real-time leaderboard with team ranks, names, college names, and round scores.
- Admin Authentication: Secure admin login using Firebase Authentication with email/password.
- Team Data Management: Enable admins to add new teams and update team scores.
- Real-time Updates: Utilize Firebase to provide real-time leaderboard updates.
- Score Calculation & Ranking: Automatically calculate total scores and rank teams based on scores, with manual override for ties.
- Data Reset: Implement a secure data reset feature with confirmation to delete all team data.

## Style Guidelines:

- Primary color: Neon purple (#D070F0) to match the TechTrix poster's theme.
- Background color: Dark navy blue (#191970) with a subtle purple/blue gradient.
- Accent color: Electric blue (#7DF9FF) for interactive elements and highlights.
- Body and headline font: 'Space Grotesk', sans-serif. A techy typeface.
- Use neon-style icons and glowing effects throughout the UI.
- Mobile-first responsive design that adapts to different screen sizes.
- Subtle animations for score updates and transitions.
